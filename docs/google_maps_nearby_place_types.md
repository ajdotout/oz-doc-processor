# Google Maps Nearby Search - Supported Place Types (Table A)

This document lists the place types supported by the Google Places API (New) for Nearby Search requests, as detailed in Table A of the official documentation. These types can be used in the `includedTypes` or `excludedTypes` parameters to filter search results.

**Note:** Types with an asterisk (*) were added as part of the November 7, 2024 release.
